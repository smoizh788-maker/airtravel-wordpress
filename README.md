# ✈️ AirTravel - Travel Booking Website

## Project Overview

AirTravel is a modern travel booking website developed using WordPress and Elementor. The website includes dynamic flight management using Custom Post Types (CPT UI) and Advanced Custom Fields (ACF).

## Technologies Used

- WordPress
- Elementor
- Astra Theme
- Astra Child Theme
- Custom Post Type UI
- Advanced Custom Fields (ACF)
- HTML
- CSS
- PHP

## Features

- Responsive Design
- Flight Search Form
- Dynamic Flight Listing
- Dynamic Single Flight Details Page
- Popular Destinations
- Newsletter Section
- Custom Header & Footer
- Book Now Button
- Mobile Friendly
